About the Database Group

This database group is affiliated with Big Data Institude (led by Prof. (Joshua) Zhexue Huang) in Shenzhen University.Currently, we have the following directions：

Efficient Algorithms for Geographic Information Systems
Efficient Algorithms for Large-scale Data Visualization
Efficient Query Processing for Spatial/Graph Databases
Distributed Algorithms for Spatial/Graph Databases
Development of Efficient Software Packages and System Prototypes

In this group, we have published many papers in top-tier conferences and journals in the database/data management field, including SIGMOD, PVLDB, ICDE, TODS, VLDBJ, and TKDE.
